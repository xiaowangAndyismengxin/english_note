**5. 语法与词汇要点**

| 项目 | 说明 | 示例/补充 |
| :--- | :--- | :--- |
| **冠词the用法口诀** | 独指方高家惯 | *We live on **the** Earth. (**独**特)*<br>*Pass me **the** book on the desk. (**指**定)*<br>*Turn to **the** left. (**方**位)*<br>*She is **the** best student. (最**高**级)*<br>***The** Smiths are coming. (**家**族)*<br> 习**惯**：*He goes to work by **the** bus. (交通工具) / She plays **the** piano beautifully. (西洋乐器) / He became **the** chairman. (头衔)* |
| **零冠词（不加the）** | | *I have **breakfast** at 7. (三餐)*<br>*He plays **football**. (球类)*<br>*She has **flu**. (大部分疾病)*<br>***Chinese** is my favorite subject. (语言)*<br>*He studies **history**. (学科)*<br>*I live in **China**, in **Shanghai**. (国家/城市)*<br>*Children **go to school**. (日常活动)* |
| **动词不规则变化** | **lie** (躺，位于) -> **lay** -> **lain** -> **lying** | *The cat **lies** on the sofa./It **lay** there all yesterday./It has **lain** there for hours./It is **lying** there now.* |
| | **lie** (撒谎) -> **lied** -> **lied** -> **lying** | *He never **lies**./He **lied** to me yesterday./He has **lied** many times./He is **lying** again.* |
| | **dig** -> **dug** -> **dug** -> **digging** | *They **dig** a hole./They **dug** a hole yesterday./They have **dug** many holes./They are **digging** now.* |
| | **mean** -> **meant** -> **meant** | *What does this word **mean**?/It **meant** something else long ago./It has always **meant** this.* |
| | **spread** -> **spread** -> **spread** -> **spreading** | *News **spreads** fast./The rumor **spread** quickly./The fire has **spread**./The virus is **spreading**.* |
| | **lead** -> **led** -> **led** | *This road **leads** to town./He **led** us to safety./Hard work has **led** to his success.* |
| **系动词** | 表示状态或变化的动词，后接形容词。 | *She **looks** happy. (状态)*<br>*The music **sounds** beautiful. (状态)*<br>*The leaves **turn** yellow in autumn. (变化)*<br>*He **became** famous. (变化)* |
| **主谓一致** | 主语由 and 连接，若指同一概念或同一人，谓语用单数。 | *Bread and butter **is** my favorite snack. (看作一种食物)*<br>*The poet and teacher **is** giving a talk. (一人兼两职)*<br>*Tom and Jerry **are** good friends. (两个人，复数)* |
| **比较级结构** | 单音节/少数双音节：比较级+and+比较级 | *The weather is getting **hotter and hotter**.*<br>*His English is becoming **better and better**.* |
| | 多音节/大部分双音节：more and more + 原级 | *The story gets **more and more interesting**.*<br>*The problem became **more and more serious**.* |
| **时间状语从句** | while 引导的从句，谓语动词必须是延续性的。 | *While I **was reading**, he came in.* (正确，read是延续动词)<br>*While I **opened** the door, he came in.* (错误，open是瞬间动词) |
| | when 引导的从句，谓语动词延续性或瞬间性均可。 | *When I **was reading**, he came in.* (正确)<br>*When I **opened** the door, he came in.* (正确) |
| **till/until** | 意为“直到……”，其中 **till** 不能用于句首。 | *I waited **till/until** he arrived.* (正确)<br>***Until** he arrived, I had waited.* (正确)<br>***Till** he arrived, I had waited.* (错误) |
| **瞬间动词进行时** | 表示即将发生的动作。 | *Hurry up! The bus **is leaving**!* |
| **Must I ... 的回答** | 否定回答用 **No, you needn't.** | *- **Must I** finish it today?* <br>*- No, you **needn't**. (You can finish it tomorrow.)* |
