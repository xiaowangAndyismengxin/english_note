**3. 连词与逻辑关系**

| 短语/句型 | 说明 | 中文释义 | 例句 |
| :--- | :--- | :--- | :--- |
| either ... or ... | 并列连词 | 要么……要么……；或者……或者…… | *You can have **either** tea **or** coffee.* |
| not only ... but (also) ... | 并列连词（**就近原则**） | 不仅……而且…… | ***Not only** my brother **but also** my parents **are** coming.* |
| as well as | 并列连词（**就远原则**） | 和；既……又…… | *My sister, **as well as** her friends, **is** going to the concert.* |
| as sb grew older | 时间状语从句引导词 | 随着某人年龄增长 | ***As she grew older**, she became wiser.* |
| as long as | 条件状语从句引导词 | 只要 | *You can borrow it **as long as** you return it.* |
| as far as | 程度/范围引导词 | 就……而言；远至…… | ***As far as I know**, the meeting is still on.* |
| as soon as | 时间状语从句引导词 | 一……就…… | *I will call you **as soon as** I arrive.* |
| as far as I am concerned | 固定表达 | 就我而言 | ***As far as I am concerned**, this is the best solution.* |
| so that | 目的状语从句引导词 | 为了，以便 | *He studied hard **so that** he could pass the exam.* |
| so as (not) to do sth | 目的状语（短语） | 为了（不）做某事 | *We left early **so as to avoid** the traffic.* |
| in order to do sth | 目的状语（短语） | 为了做某事 | ***In order to catch** the train, we left early.* |
| until then | 时间状语 | 直到那时 | *I won’t know the result **until then**.* |
| long before | 时间状语 | 在……很久以前 | *He had left **long before** we arrived.* |
| let alone | 更不用说 | 更别提…… | *He can’t run a mile, **let alone** a marathon.* |
| besides | 介词/副词（**表追加**） | 除……之外（还有） | ***Besides** English, she speaks French.* |
| except (for) | 介词（**表排除**） | 除了…… | *Everyone passed **except (for)** Tom.* |
| what’s worse | 副词性短语（**表递进**） | 更糟的是 | *It rained all day, and **what’s worse**, I forgot my umbrella.* |
| furthermore | 副词（**表递进**） | 此外，而且 | *The plan is efficient. **Furthermore**, it is cost-effective.* |
| no longer | 副词短语 | 不再 | *He **no longer** lives here.* |

**5. 数字、年龄与序数词读法**

| 项目 | 读法规则/例子 | 中文释义 | 例句/说明 |
| :--- | :--- | :--- | :--- |
| **小数** | 0.25 | 零点二五 | *读作: **(zero) point two five*** |
| | 2.5 | 二点五 | *读作: **two point five*** |
| **分数** | 1/2 | 二分之一 | *读作: **a half** 或 **one half*** |
| | 1/5 | 五分之一 | *读作: **one fifth** (分子基数词，分母序数词)* |
| | 2/3 | 三分之二 | *读作: **two thirds** (分子>1，分母加s)* |
| **年龄表达** | He is 18 years old. | 他18岁。 | ***He is an 18-year-old boy.*** |
| | He is a boy of 18. | 他是个18岁的男孩。 | ***He is in his late teens.*** |
| **序数词** | 12th (twelfth) | 第十二 | *特殊变化* |
| | 9th (ninth) | 第九 | *去e加th* |
| | 20th (twentieth) | 第二十 | *规律：y -> ieth* |
| | 30th (thirtieth) | 第三十 | *规律：y -> ieth* |
