**📘 第一学期最终版英语笔记：短语、句型与语法（含例句）**

**1. 动词搭配与句型**

| 短语/句型 | 词性/说明 | 中文释义 | 例句 |
| :--- | :--- | :--- | :--- |
| make ... do ... | 使役动词+动词原形 | 使/让……做…… | *The loud noise **made** the baby **cry**.* |
| look forward to doing | 动词短语 | 期待做某事 | *I **look forward to visiting** Paris next year.* |
| practice doing | 动词短语 | 练习做某事 | *She **practices playing** the piano every day.* |
| mind doing | 动词短语 | 介意做某事 | *Would you **mind closing** the window?* |
| start to do / start doing | 动词短语 | 开始做某事 | *It **started to rain**. / He **started learning** Spanish.* |
| begin to do / begin doing | 动词短语 | 开始做某事 | *The movie **begins to get** exciting. / Let’s **begin writing**.* |
| stop doing | 动词短语 | 停止正在做的事 | *Please **stop talking** during the lecture.* |
| stop to do | 动词短语 | 停下来去做另一件事 | *He **stopped to tie** his shoelaces.* |
| continue to do sth | 动词短语 | 继续做某事 | *They **continued to work** despite the difficulty.* |
| decide to do sth | 动词短语 | 决定做某事 | *We **decided to order** pizza for dinner.* |
| determine to do sth | 动词短语 | 决心做某事 | *She **is determined to pass** the final exam.* |
| help sb (to) do sth | 动词短语 | 帮助某人做某事 | *Can you **help me (to) carry** this box?* |
| help sb with sth | 动词短语 | 帮助某人用某物 | *I **helped her with** the book.* |
| ask sb to do sth | 动词短语+动词原形 | 要求某人做某事 | *The teacher **asked the students to be** quiet.* |
| wake up sb / wake sb up | 动词短语，通常代词放中间 | 叫醒某人 | *Please **wake me up** at 7 a.m. tomorrow.* |
| show sb sth / show sth to sb | 动词短语 | 给某人看某物 | *He **showed me** his new phone. / He **showed his new phone to me**.* |
| explain to sb | 动词短语 | 向某人解释 | *Could you **explain** this rule **to me** again?* |
| pass sth to sb | 动词短语 | 把某物递给某人 | *Please **pass the salt to your father**.* |
| spend time (in) doing sth | 动词短语 | 花费时间做某事 | *I **spent two hours (in) finishing** my homework.* |
| spend money on sth | 动词短语 | 花钱在某物上 | *They **spent a lot of money on** their new car.* |
| learn about | 动词短语 | 学习关于…… | *In history class, we **learned about** ancient Rome.* |
| take its name from | 动词短语 | 得名于…… | *The city **takes its name from** a nearby river.* |
| make a list of | 动词短语 | 列出…… | *Before shopping, I **made a list of** things I need.* |
| realize one's dream | 动词短语 | 实现梦想 | *After years of hard work, she finally **realized her dream**.* |
| dream of doing sth | 动词短语 | 梦想做某事 | *He **dreams of becoming** an astronaut.* |
| have interest in | 动词短语 | 对……有兴趣 | *Do you **have any interest in** classical music?* |
| get/be curious about | 动词短语 | 对……好奇 | *The child **is curious about** how airplanes fly.* |

**2. "be" 动词与介词短语**

| 短语 | 中文释义 | 例句 |
| :--- | :--- | :--- |
| be good at / do well in | 擅长…… | *My brother **is good at** mathematics. / He **does well in** science.* |
| be bad at / weak in doing sth | 不擅长…… | *I **am bad at** remembering names. / She **is weak in** writing essays.* |
| be honest with | 对……诚实 | *You should always **be honest with** your friends.* |
| be important for/to | 对……重要 | *Sleep **is important for** your health. / This letter **is important to** me.* |
| be surprised at sth | 对某事感到惊讶 | *We **were surprised at** the sudden news.* |
| be busy with | 忙于（某事） | *Mom **is busy with** preparations for the party.* |
| be busy doing sth | 忙于做某事 | *He **is busy preparing** for his upcoming trip.* |
| be full of / be filled with | 充满…… | *The room **was full of** laughter. / The glass **is filled with** water.* |
| be different from | 与……不同 | *Life in the countryside **is very different from** that in the city.* |
| be similar to | 与……相似 | *Your bag **is similar to** mine.* |
| be famous for | 因……而出名 | *Italy **is famous for** its delicious food.* |
| be curious about | 对……好奇 | *Tourists **are often curious about** local customs.* |
| be determined to do sth | 决心做某事 | *She **is determined to improve** her English skills.* |
| be good for | 对……有益 | *Eating vegetables **is good for** your body.* |

**3. "It" 形式主语句型**

| 句型 | 中文释义 | 说明 | 例句 |
| :--- | :--- | :--- | :--- |
| It is adj. to do sth. | 做某事是……的。 | 形容词描述事 | ***It is wonderful to travel** around the world.* |
| It is adj. for sb to do sth. | 对某人来说，做某事是……的。 | 形容词描述事（如 easy, difficult） | ***It is easy for him to solve** this math problem.* |
| It is adj. of sb to do sth. | 某人做某事是……的。 | 形容词描述人（如 kind, clever） | ***It was very kind of you to help** me.* |
| It takes sb. time to do sth. | 做某事花费某人多少时间。 | | ***It takes me 30 minutes to walk** to school.* |

**5. 语法与词汇要点**

| 项目 | 说明 | 示例 | 例句 |
| :--- | :--- | :--- | :--- |
| **不规则动词** | spread - spread - spread | 原形 - 过去式 - 过去分词 | *She **spreads** butter on the bread. / Yesterday she **spread** butter on the bread. / She has **spread** butter on the bread.* |
| **情态动词** | should (过去式) - shall (现在式) | | *You **should** see a doctor. / **Shall** we go now?* |
| **不可数名词** | advice（建议）, trade（贸易） | 前面不能用 `a/an`，没有复数形式 | *Thank you for your good **advice**. / International **trade** is important for the economy.* |
| **复合不定代词** | 默认为单数 | Everyone **is** here. | ***Somebody is** waiting for you at the door.* |
| **oo发音口诀** | 字母“k”前不能长，好脚站木羊毛短 | 短音：**f**oo**t**, g**oo**d, st**oo**d, w**oo**d, w**oo**l<br>长音：f**oo**d, m**oo**n, r**oo**m | *Look at your **foot**. / I love the **food** here.* |
| **词汇** | modernity (n.) 现代性 | | *The building is a symbol of **modernity**.* |
| | guidance (n.) 指导 | | *I need some **guidance** on how to use this software.* |
| | a guide to... ……的指南 | | *This book is **a guide to** Chinese cooking.* |
| | tour (n./v.) 旅行/游览 | | *We went on a **tour** of the museum. / They will **tour** Europe next summer.* |
| | botanic (adj.) 植物的 | | *We visited the **botanic** garden.* |
| | formal (adj.) 正式的 | | *You need to wear **formal** clothes to the ceremony.* |
| | roast (v./adj.) 烤/烤的 | | *Let's **roast** some chicken for dinner. / I'd like **roast** beef.* |
| | agent (n.) 工作人员，代理人 | | *She is a travel **agent**.* |
| | museum (n.) 博物馆 | | *The **museum** has a great collection of art.* |
| | similarity (n.) 相似性 | | *There is a strong **similarity** between the two stories.* |
| | explanation (n.) 解释 | | *Can you give me an **explanation** for your absence?* |
| | impatient (adj.) 不耐烦的 | | *He became **impatient** with the long wait.* |
| | inexpensive (adj.) 便宜的 | | *This restaurant serves good and **inexpensive** food.* |
| | incorrect (adj.) 不正确的 | | *Your answer is **incorrect**.* |
| | leave (v.) 留下 | | *I **left** her a message.* |
